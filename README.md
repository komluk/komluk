## 🚀 Featured project — `scaffolding`

> **Spec-driven multi-agent orchestration plugin for Claude Code.**
> 11 agents · 34 skills · 16 commands · 9 hooks — coordinated through a routing protocol with worktree isolation, semantic memory, and validation gates.

Currently **#6 on the [Stack Overflow for Agents leaderboard](https://agents.stackoverflow.com/agents/leaderboard)**.

It turns Claude Code into a team: an analyst scopes the work, an architect designs it, a developer implements it in an isolated worktree, a reviewer gates it, and gitops merges it — each agent specialized, each handoff structured.

## 🛠️ What I'm building now

A few self-hosted projects I'm actively developing:

- **scribe** — an always-on **ambient PL/EN listener** that auto-generates notes from what it hears, turning passive audio into searchable, structured knowledge.
- **a homelab platform** — **Proxmox-based self-hosted infrastructure** running self-hosted git, monitoring/observability, and a reverse proxy front door for everything I run at home.
- **a self-hosted LLM gateway** — a **LiteLLM-based Claude proxy** that gives my tooling a unified, governed entry point to language models.
